# 功能实现

- 注册时,能够在**前端**进行格式判断并给出实时反馈,若输入数据不合法,会有invalid字样在右侧显示,
  一旦合法,invalid字样消失
- 只有当数据均合法时,注册按钮才可以点击
- 前端可以阻止提交不合法数据
- 服务器端可以检测出重复用户名\邮箱\手机\学号,在提交时通过弹窗给出反馈
- 若成功注册,注册后显示用户详情,通过return按钮可以返回
- 输入形如http://localhost:8000?username=abc 时，abc是已注册用户，显示“详情”，否则回到注册页面
